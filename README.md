# Fire Detection using Deep Learning

## Problem
Fire accidents can cause severe damage. Early detection is crucial for safety.

## Approach
Developed a deep learning-based image classification model to detect fire from images.

## Technologies Used 
- Python
- TensorFlow / Keras
- OpenCV
- Numpy, Pandas

## Results 
- Model trained to classify dire vs non-fire images
- Achieved basic prediction capability on test data

## Future Improvements
- Real-time video detection
- Web-basede interface for alerts
